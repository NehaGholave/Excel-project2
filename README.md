## 📊 HR Analytics Dashboard | Excel + Power Query Project
This project presents an interactive HR Analytics Dashboard designed using Excel and Power Query to analyze employee turnover, workforce demographics, and organizational health. The dashboard provides a comprehensive view of attrition patterns to help HR professionals develop better retention strategies.

## ❓ Business Problem
High employee attrition can be costly and disruptive for any organization. This dashboard was developed to identify the root causes of turnover—analyzing whether attrition is driven by specific job roles, age groups, or educational backgrounds—enabling data-driven decisions to improve employee satisfaction and stability.

## 🚀 Project Highlights
- End-to-End Development: Built a complete analytical tool from raw employee data.

- Automated Data Processing: Utilized Power Query for efficient data cleaning and transformation.

- Interactive UX: Integrated slicers for real-time filtering by department and education field.

- Strategic Insights: Focused on Key Performance Indicators (KPIs) that impact business continuity.

## 📊 Key Features
## 📈 High-Level KPIs:

- Total Employees: Snapshot of the total workforce size.

- Active Employees: Count of current staff members excluding those who left.

- Attrition Rate: Percentage of turnover relative to the total headcount.

- Average Age: General workforce demographic baseline.

⭐ Job Satisfaction Rating: Visual representation of employee sentiment based on internal surveys.

🚻 Gender-wise Distribution: Visual breakdown of the male and female workforce balance.

## 📌 Insights & Visualizations
🎓 Education-Wise Attrition: Identifies which educational backgrounds (Life Sciences, Medical, etc.) see the highest turnover volume.

🏢 Department-Wise Attrition: A pie chart breakdown revealing which departments (R&D, Sales, HR) are experiencing the most departures.

🛠️ Attrition by Job Role: A treemap highlighting specific roles—such as Laboratory Technicians and Sales Executives—as high-turnover positions.

📅 Attrition by Age Group: Analysis showing how turnover varies across different life stages, particularly within the younger workforce.

💍 Attrition by Marital Status: Funnel chart showing how turnover correlates with personal demographic categories.
